## 나중에 (Blaze 유료 전환 필요) — 미정리, 보류 중

### 1. 자동 데이터 정리 (Cloud Functions + Cloud Scheduler)
- 만료된 nw_calls 문서 자동 삭제 (cron, 예: 1시간마다 expiresAt 지난 것 제거)

### 2. POS API 연동 모듈 (사용자가 이전에 요청, 보류)
- POS Webhook 수신 → availableTables 실시간 업데이트 (Cloud Functions HTTP 엔드포인트)
- 자동 디스패치 응답 (Auto-Responder) — 빈 테이블 있으면 점주 조작 없이 자동 'ok' 응답
- 테이블 선점 Lock — 매칭 확정 시 POS 가주문 API 호출 (Axios)
- 노쇼 시 Lock 해제 — Cloud Tasks/Cron 스케줄러
- 현재: 데모 매장 자동응답(클라이언트 setTimeout)으로 흉내만 냄

### 3. 노쇼 신뢰점수 시스템 (부분적으로 무료 가능, 본격은 서버)
- 노쇼 누적 카운트 → 상습 노쇼자 이용 제한/차단
- 신뢰점수 기반 차등 (점수 낮으면 선입금 요구 등)
- 현재: 노쇼 1회 = 30분 패널티 단발성만

### 4. 선입금/결제 (PG 연동, 큰 작업)
